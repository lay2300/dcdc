A romantic interactive web application featuring a lock screen, memory slideshow, and surprise reveal.

## Features

- **Lock Screen**: Requires a passcode to enter (Default: `1166`).
- **Countdown**: A suspenseful countdown timer.
- **Memory Lane**: A slideshow of photos with captions.
- **Surprise Reveal**: A final card reveal with confetti and sound effects.
- **Background Effects**: Floating hearts and background music support.

## Setup

1. Place your images (`2.jpg`, `3.jpg`, `4.jpg`) in the project folder.
2. Ensure sound effect files are present (`sfxClick`, `sfxSuccess`, `sfxError`, `sfxTick`, `sfxShutter`, `sfxPop`, `bgMusic`).
3. Open `index.html` in a web browser.

## Customization

- Modify `script.js` to change the passcode or add more memories to the `memories` array.

## Troubleshooting

- If images do not load, please ensure all filenames are in lowercase (e.g., `2.jpg`, `card.jpg`).
